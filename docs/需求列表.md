# 需求列表

## 1. 页面导航需求
- 实现导航栏切换功能，点击导航链接显示对应内容区域
- 保持导航状态，高亮当前活跃页面
- 支持通过卡片链接跳转到对应功能页面

## 2. 剪贴板监听需求
- 实时监听剪贴板内容变化
- 自动识别抖音和TikTok的链接格式
- 在账号管理和链接下载页面自动填充对应输入框
- 处理剪贴板访问权限问题

## 3. 链接下载需求
- 支持批量下载多个链接（按换行分隔）
- 支持抖音和TikTok两个平台
- 支持设置代理服务器
- 显示下载进度和状态信息
- 处理下载失败的情况，显示错误信息
- 显示下载成功后的保存路径

## 4. 账号管理需求
- 支持批量添加多个账号（按换行分隔）
- 支持编辑账号信息（账号名称、用户主页链接或Sec User ID）
- 支持删除账号
- 支持下载账号的不同类型作品（发布、喜欢、收藏）
- 显示账号下载状态（开始下载、作品列表、完成数量、完成、失败）
- 支持表格列宽调整
- 显示账号主页链接，支持点击打开

## 5. 下载设置需求
- 支持自定义保存路径
- 支持选择不同下载质量
- 支持调整下载线程数
- 支持启用/禁用按文件夹归档
- 支持从不同浏览器获取Cookie
- 支持删除Cookie
- 保存设置到本地存储和后端服务器
- 加载设置时优先使用后端数据，失败时使用本地存储数据
