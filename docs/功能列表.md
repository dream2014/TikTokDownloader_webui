# 功能列表

## 1. 导航功能
- 首页导航
- 链接下载页面导航
- 账号管理页面导航
- 下载设置页面导航

## 2. 剪贴板监听功能
- 实时监听剪贴板内容
- 自动识别并填充抖音/TikTok链接到对应输入框

## 3. 链接下载功能
- 支持多行输入链接（每行一个链接）
- 选择平台（抖音/TikTok）
- 支持代理设置（可选）
- 显示下载进度和状态
- 处理下载失败的情况

## 4. 账号管理功能
- 添加账号（支持多行输入用户主页链接或Sec User ID）
- 编辑账号信息
- 删除账号
- 下载账号作品（支持选择作品类型：发布、喜欢、收藏）
- 显示账号列表和下载状态
- 表格列宽拖动调整

## 5. 下载设置功能
- 保存路径设置
- 下载质量设置（高、中、低）
- 下载线程数设置（1-10）
- 按文件夹归档设置
- Cookie管理（从浏览器获取、删除）
- 浏览器选择（Chrome、Edge、Firefox、Brave、Opera、Vivaldi）
- 设置保存到本地和后端
